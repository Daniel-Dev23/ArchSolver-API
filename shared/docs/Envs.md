# **Variables de Entorno**

En esta sección, se centralizan las diferentes recomendaciones a seguir, para lograr una correcta implementación de las variables de entorno del sistema.

| Variable   | Requerido | Default      | Concepto                                                         | Notación | 
| ---------- | --------- | ------------ | ---------------------------------------------------------------- | -------- |
| `NODE_ENV` | Si        | development  | Define el tipo de ambiente que estara interpretando el servidor. | `development` ó `production` |
| `VERSION`  | Si        | v0.1.0-alpha | Define el estado de maduración del producto.                     | `1.0.0-alpha` ó `1.0.0-beta` ó `1.0.0-stable` |
| `APP_PORT` | Si        | 80           | Define el número de puerto del servidor **App**.                 | `80`, `81`, `82`, etc. |
